# SQL Files Documentation

## Overview
This document provides comprehensive documentation for all SQL files included in the repository.

## SQL Files

### 1. create_table.sql
   - **Description:** Script for creating the necessary tables in the database.
   - **Usage:** Execute this script first to set up the database schema.

### 2. insert_data.sql
   - **Description:** Inserts sample data into the previously created tables.
   - **Usage:** Run this script after creating tables to populate the database with initial data.

### 3. update_records.sql
   - **Description:** Used to update existing records in the database.
   - **Usage:** Execute this script when any modifications to current records are required.

### 4. delete_records.sql
   - **Description:** Deletes specified records from the database.
   - **Usage:** Use this script to remove unwanted records from the tables.

### 5. select_queries.sql
   - **Description:** Various select queries for data retrieval.
   - **Usage:** This script is used to fetch different sets of data based on requirements.

## Conclusion
This documentation should help users understand the purpose and usage of each SQL file in this repository.
